Step 2: Feature Extraction - Implementation

Python Code

# feature_extraction.py import numpy as np class ProteinFeatureExtractor: AMINO_ACIDS = 'ACDEFGHIKLMNPQRSTVWY' def compute_aa_composition(self, seq): """Calculate amino acid frequencies""" aa_counts = {aa: 0 for aa in self.AMINO_ACIDS} for aa in seq: if aa in aa_counts: aa_counts[aa] += 1 # Convert to frequencies frequencies = [aa_counts[aa] / len(seq) for aa in self.AMINO_ACIDS] return np.array(frequencies) def compute_dipeptide_composition(self, seq): """Calculate dipeptide frequencies""" dipeptides = [a1+a2 for a1 in self.AMINO_ACIDS for a2 in self.AMINO_ACIDS] dipeptide_counts = {dp: 0 for dp in dipeptides} for i in range(len(seq) - 1): dp = seq[i:i+2] if dp in dipeptide_counts: dipeptide_counts[dp] += 1 frequencies = [dipeptide_counts[dp] / (len(seq) - 1) for dp in dipeptides] return np.array(frequencies)

Example Calculation

Sequence: "AAAAACCCGGG" (length=11) # Amino Acid Composition A = 5/11 = 0.45 (45%) C = 3/11 = 0.27 (27%) G = 3/11 = 0.27 (27%) # Dipeptide Composition Pairs: AA,AA,AA,AA,AC,CC,CC,CG,GG,GG AA = 4/10 = 0.40 (40%) AC = 1/10 = 0.10 (10%) CC = 2/10 = 0.20 (20%)

Step 3: Feature Selection - Implementation

Python Code

# feature_selection.py from sklearn.feature_selection import VarianceThreshold class FeatureSelector: def variance_threshold_selection(self, X, threshold=0.01): """Remove low-variance features""" print(f"Original features: {X.shape[1]}") selector = VarianceThreshold(threshold=threshold) X_selected = selector.fit_transform(X) mask = selector.get_support() print(f"Selected features: {X_selected.shape[1]}") print(f"Removed: {X.shape[1] - X_selected.shape[1]}") return X_selected, mask # Usage selector = FeatureSelector() X_selected, mask = selector.variance_threshold_selection(X, 0.01)

Selection Logic

# Example: Remove features with variance < 0.01 Feature 1: [0.01, 0.01, 0.01, 0.01] Variance = 0.000 → REMOVE ✗ Feature 2: [0.01, 0.08, 0.15, 0.03] Variance = 0.032 → KEEP ✓ Feature 3: [0.05, 0.04, 0.05, 0.05] Variance = 0.0002 → REMOVE ✗

Conclusions & Key Findings

Main Conclusions

1. Simple Features Achieve High Accuracy
• Amino acid + dipeptide composition: 96% accuracy
• No complex feature engineering required
• No deep learning necessary
2. Feature Selection is Crucial
• Improved accuracy: +2-4% across all classifiers
• Reduced training time: 40-72% faster
• Dimensionality reduction: 79.3% fewer features
• Enhanced interpretability
3. Random Forest Outperforms
• Highest accuracy: 96.0%
• Most consistent: 95.6% ± 1.5% (CV)
• Provides feature importance rankings
• Robust across validation folds
4. Amino Acids More Important
• 90% of AA features retained
• Only 17% of dipeptide features retained
• Single amino acid frequencies highly informative
